＃＃遊び方
1実行すると対象文字、表示文字が出力される
2対象文字と表示文字の差をはじめに回答する
3正解ならば欠損している文字を出力内容にしたがって入力する
4不正解ならば不正解という出力とともに2からくり返される
5正解ならば終了

＃プログラム内の解説*
 main関数：クイズプログラムの全体の流れを担当する．* shutudai関数：ランダムに選んだ問題を出題し，解答をmain関数に返す．
 kaitou関数：回答と正解をチェックし，結果を出力する．
